About this Project

A simple and efficient product management system

Description

Product Management is a web application built with Django that allows users to register, view, edit and delete products in a simple and organized way. It includes user authentication so each account can manage its own inventory securely.

Features
  • User registration and login
  • Create, edit and delete products
  • Product detail view
  • Responsive design with Bootstrap
Built With
Django Bootstrap SQLite Python
